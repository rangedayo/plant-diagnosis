# GT 재검 워크리스트 — R13 Arm C FP14 (방향 a)

> 목적: is_healthy FP 14건 중 GT="건강"이 방어 가능한지 **사람이 이미지로 재검**.
> 원칙(§8.1): **라벨 판단은 사람만.** 아래 "모델 증상"은 모델의 *주장*이지 사실이 아님(과대 보고 가능) → **반드시 실제 이미지와 대조.**
> 이미지 경로: `test_data/main_eval/images/<image_id>.jpg`

## 어느 쪽으로 결론나도 정보가 된다 (프레이밍)
각 케이스에서 "모델이 보고한 병리 징후(괴사·반점·황화)"가 —
- **실제 이미지에 있다** → GT를 ambiguous/비건강으로 옮길 근거(= 그 FP는 애초에 진짜 오류가 아니었음). 동시에 모델 지각이 정확하다는 뜻.
- **이미지에 없다(모델 과대 보고)** → GT="건강" 유지(= 진짜 FP). 동시에 **모델이 심각도를 과장**한다는 증거 → 방향 (b)의 근거가 됨.

→ 그래서 이 재검은 (b)를 할지 말지도 같이 판정해 줍니다.
