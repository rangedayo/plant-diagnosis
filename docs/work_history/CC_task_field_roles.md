## 2. 규칙 — 두 근거 사슬 분리

**핵심: 종명은 *형태*로 뒷받침(요약), 건강 상태는 *증상*으로 뒷받침(판단 근거). 절대 교차 금지.**

**[진단 요약] (summary) = 종명 식별(형태 근거) + 건강 상태 한 줄**
- 구조: "[잎 모양·생장형·색·무늬 등 형태적 특징]으로 보아 [종명]으로 보입니다. [건강 상태: 병해가 의심됩니다 / 건강해 보입니다 / 관리가 필요해 보입니다]."
- ❌ 증상(반점·갈변·마름)으로 종명을 뒷받침하지 말 것 (증상은 종 식별 근거 아님).
- ❌ 증상 관찰 문장을 요약의 건강-근거로 넣지 말 것 → 그건 cause로.

**[이렇게 판단했어요] (cause) = 건강 진단 근거(증상 근거)**
- 구조: "관찰된 [증상]으로 보아 [건강 상태 근거 / 원인 추정]."
- 경미: "관찰된 [증상]은 이 종에서 흔히 나타날 수 있는 경미한 손상으로 보이며 반드시 병을 의미하지는 않습니다." (겸손 유지)
- 건강: "관찰된 이상이 없어 특별한 원인은 보이지 않습니다." (긍정 유지)

## 3. 취지 예시 (구조 — 실제 형태/증상은 모델이 채움)

**epipremnum_005 (병해):**
- 요약: "넓고 광택 있는 잎 형태와 짙은 녹색으로 보아 필로덴드론 도메스티쿰으로 보입니다. 잎 상태로 보아 병해가 의심됩니다."
- 판단 근거: "관찰된 잎의 불규칙한 흑갈색 반점과 가장자리 갈변으로 보아 병해가 의심됩니다."
- (지금: 증상-근거가 요약에 들어가 있음 → cause로 이동, 요약엔 형태→종명 추가)

**chlorophytum_002 (건강):**
- 요약: "가늘고 긴 선형 잎이 모여 자라는 형태로 보아 디아넬라 카에룰레아일 가능성이 있습니다. 전반적으로 건강해 보입니다."
- 판단 근거: "관찰된 일부 잎끝 국소적 갈변은 이 종에서 흔히 나타나는 자연스러운 현상으로, 건강 이상 신호로 보기는 어렵습니다." (이미 좋음 — 유지)
- (지금: 요약이 "국소적 갈변으로 보아 디아넬라" = 증상으로 종명 뒷받침 → 형태로 교체)

**sansevieria_001 (건강):**
- 요약: "두껍고 직립한 잎과 얼룩무늬로 보아 산세베리아로 보입니다. 관찰된 이상이 없어 건강한 상태로 보입니다."
- 판단 근거: "현재 관찰된 이상이 없어 특별한 원인은 보이지 않습니다."
- (지금: 요약에 종명 근거가 아예 없음 → 형태→종명 추가)
