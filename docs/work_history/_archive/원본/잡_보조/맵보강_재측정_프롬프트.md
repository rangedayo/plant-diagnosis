## 추가할 별칭 (4개) — 명백히 같은 식물만

**그룹 A — 학명 표기/종소명 차이:**
- `Spathiphyllum wallisii` → 스파티필름 (맵엔 종소명 없는 Spathiphyllum만 있었음)
- `Aglaonema modestum` → 아글라오네마 (속이 맵에 아예 없었음)
- `Aglaonema commutatum` → 아글라오네마

**그룹 B — 분류학적 재분류 (옛이름=새이름):**
- `Dracaena trifasciata` → 산세베리아 (산세베리아의 현재 학명. Sansevieria trifasciata와 동일 식물)

**추가하지 않음 (모델의 실제 오답 — 정직하게 남김):**
- Dracaena fragrans→행운목(기존 유지), Dracaena braunii, Phalaris arundinacea, Costus arabicus 등은
  완전히 다른 종이므로 별칭 추가 금지.

**예상 부작용 (의도된 정확한 채점):**
- self_haengun_004는 모델이 Dracaena trifasciata로 출력 → 산세베리아로 매핑 → 정답(행운목)과 불일치 → wrong.
  이는 "행운목을 산세베리아로 잘못 본" 모델의 진짜 오답을 정확히 표시하는 것. 정상.
